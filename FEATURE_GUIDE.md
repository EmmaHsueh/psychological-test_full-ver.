# 每日推薦測驗功能使用指南

## 🎉 新功能介紹

我已經成功為你的心理測驗網站實作了**每日推薦測驗**功能！這個功能使用進階推薦演算法，會根據你的歷史記錄來推薦最適合你的測驗。

---

## ✨ 功能特色

### 1. 智能推薦演算法
- **用戶偏好分析** (35%權重)：分析你最常做的測驗類別和標籤
- **測驗熱度** (25%權重)：優先推薦熱門測驗
- **時間新鮮度** (20%權重)：避免重複推薦最近做過的測驗
- **主題多樣性** (20%權重)：確保推薦的測驗類別多元化

### 2. 豐富的測驗數據庫
包含 **20+ 精選心理測驗**，涵蓋8大類別：
- 👤 人格分析 (MBTI、九型人格、大五人格、DISC等)
- 💼 職涯發展 (職業性格、優勢測驗等)
- ❤️ 情緒智商 (EQ測驗、同理心測驗等)
- 🧠 思維模式 (左右腦、學習風格、創造力等)
- 💕 戀愛關係 (愛情風格、依附風格等)
- 🎮 趣味娛樂 (色彩心理學、IQ測驗、記憶力等)
- 🌿 心理健康 (壓力測驗、社交焦慮、幸福指數等)
- ⭐ 價值觀 (核心價值觀、道德陣營等)

### 3. 獨立推薦頁面
- **今日精選推薦**：每日一個特別推薦的測驗，配有精美大卡片
- **更多推薦測驗**：一次顯示6個推薦測驗供你選擇
- **推薦理由**：清楚說明為什麼推薦這個測驗給你
- **個人統計**：顯示你的測驗歷程

## 📱 如何使用

### 步驟1：進入推薦頁面
1. 進入「測驗選擇」頁面
2. 點擊右上角的「為你推薦」按鈕

### 步驟2：查看每日推薦
- 頁面頂部會顯示今日精選推薦
- 包含測驗名稱、描述、類別、預估時間
- 推薦理由會告訴你為什麼適合你

### 步驟3：探索更多推薦
- 向下滑動查看更多推薦測驗
- 每個測驗卡片都有清楚的說明
- 點擊「開始測驗」即可前往測驗頁面

### 步驟4：追蹤統計
- 頁面底部顯示你的測驗統計
- 查看已完成測驗數量、最喜歡的類別等

## 🔄 推薦演算法工作原理

### 新用戶
- 推薦熱門測驗
- 鼓勵嘗試不同類別
- 基於測驗的受歡迎程度排序

### 有歷史記錄的用戶
1. **分析你的偏好**
   - 計算你最常做的測驗類別
   - 分析你偏好的測驗標籤

2. **避免重複**
   - 7天內做過的測驗不會再推薦
   - 最近做過的類別會降低權重

3. **保持多樣性**
   - 即使你喜歡某類別，也會推薦其他類型
   - 確保你能探索不同主題的測驗

4. **每日變化**
   - 使用日期作為種子
   - 確保每天推薦不同的測驗
   - 同一天多次訪問會看到相同推薦

## ❓ 常見問題

**Q: 為什麼每天推薦都一樣？**
A: 這是設計行為！同一天的推薦保持一致，隔天會自動更新。

**Q: 如何獲得更準確的推薦？**
A: 多完成不同類別的測驗，系統會更了解你的偏好。

**Q: 可以跳過今日推薦嗎？**
A: 當然！向下滑動查看更多推薦，或直接去「測驗選擇」頁面。

**Q: 推薦的測驗會重複嗎？**
A: 7天內做過的測驗不會再推薦，但7天後可能會再次出現。
